Exemplo de utilização de variáveis CSS

Este é um exemplo de como utilizar variáveis CSS para definir valores que podem ser reutilizados em diferentes partes do seu estilo. Neste exemplo, estamos utilizando variáveis para definir a cor principal e o tamanho da fonte do texto.

Também estamos utilizando a variável de cor secundária para definir a cor de fundo do body. Tudo isso pode ser facilmente ajustado modificando as variáveis no :root.
